Aula 5: Instalando Pacotes e Permissões

Nesta aula, você vai aprender:

  • Como instalar programas com apt
  • Gerenciar permissões com chmod
  • Executar comandos como superusuário (sudo)

Instalando Softwares

No Linux, usamos gerenciadores de pacotes para instalar programas de forma segura e fácil.

sudo & apt

SuperUser DO e Advanced Package Tool.

$ sudo apt update
$ sudo apt install python3

O sudo dá permissões de administrador temporariamente.

Permissões (chmod)

Controla quem pode ler, escrever ou executar arquivos.

$ chmod +x script.sh

+x torna o arquivo executável.

Scripts

Automatizando tarefas com Shell Script.

$ echo "echo 'Olá Mundo'" > script.sh
$ chmod +x script.sh
$ ./script.sh
Olá Mundo
