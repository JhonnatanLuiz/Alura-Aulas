O que é o Terminal?

O terminal é uma interface de texto que permite interagir diretamente com o sistema operacional. Ao contrário da interface gráfica (GUI), onde você clica em ícones, no terminal você digita comandos para executar tarefas.

Comandos Básicos de Navegação

pwd

Print Working Directory - Mostra onde você está.

$ pwd
/home/usuario

ls

List - Lista arquivos e pastas.

$ ls
Documentos Imagens Downloads projeto

cd

Change Directory - Muda de pasta.

$ cd Documentos
$ pwd
/home/usuario/Documentos
