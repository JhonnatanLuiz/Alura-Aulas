Aula 3: Editando Arquivos com VI e Nano

Nesta aula, você vai aprender:

  • Como usar o editor VI
  • Como usar o editor Nano
  • Filtrar e ordenar conteúdo de arquivos

Editores de Texto no Terminal

Muitas vezes você precisará editar arquivos de configuração diretamente no servidor, onde não há interface gráfica. Para isso, usamos editores de terminal.

Nano

O editor mais simples para iniciantes.

$ nano arquivo.txt
  • Ctrl + O: Salvar (Write Out)
  • Ctrl + X: Sair

VI / VIM

Poderoso e onipresente, mas com curva de aprendizado.

$ vi arquivo.txt
  • i: Modo de inserção (para digitar)
  • Esc: Sair do modo de inserção
  • :w: Salvar
  • :q: Sair
  • :wq: Salvar e sair

Visualizando Conteúdo

Comandos úteis para ler arquivos grandes.

$ head arquivo.txt # Primeiras 10 linhas
$ tail arquivo.txt # Últimas 10 linhas
$ sort nomes.txt # Ordena alfabeticamente
