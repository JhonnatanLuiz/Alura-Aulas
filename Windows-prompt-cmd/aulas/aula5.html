📦 Winget - Gerenciador Oficial

O Winget é o gerenciador de pacotes oficial da Microsoft, já incluído no Windows 11 e disponível para Windows 10!

✅ Vantagens

  • • Oficial da Microsoft
  • • Já vem instalado
  • • Interface simples
  • • Grande catálogo

🔄 Comandos

  • • winget install
  • • winget upgrade
  • • winget search
  • • winget list
Exemplos Winget 
# Buscar programa
C:\> winget search "visual studio code"

# Instalar
C:\> winget install Microsoft.VisualStudioCode

# Atualizar tudo
C:\> winget upgrade --all

# Ver instalados
C:\> winget list

💻 Terminais Modernos

🎨 Windows Terminal

Terminal oficial com abas, temas e customização

winget install Microsoft.WindowsTerminal

🚀 Cmder

Terminal portátil com Git integrado

choco install cmder -y

🎨 Personalização do Windows Terminal

  • • Temas: Clique na seta ▼ → Configurações → Aparência
  • • Fontes: Instale "Cascadia Code" ou "Fira Code"
  • • Transparência: Ajuste opacidade do fundo
  • • Cores: Escolha esquemas pré-definidos
Instalar fonte Cascadia Code 
C:\> winget install Microsoft.CascadiaCode

🐧 WSL - Linux no Windows

O Windows Subsystem for Linux (WSL) permite executar Linux nativamente no Windows!

💡 Por que usar? Acesse ferramentas Linux, desenvolva aplicações, aprenda comandos Unix!

Instalar WSL (como Admin) 
# Instalar WSL com Ubuntu (padrão)
C:\> wsl --install

# Listar distribuições disponíveis
C:\> wsl --list --online

# Instalar distribuição específica
C:\> wsl --install -d Ubuntu-22.04

# Entrar no Linux
C:\> wsl

⚠️ Nota: Reinicie o computador após a instalação!

🐧 Comandos Básicos do Linux

📁 ls

Listar arquivos

ls -la

📂 cd

Mudar diretório

cd /home

📝 mkdir

Criar pasta

mkdir projetos

📍 pwd

Ver diretório atual

pwd

📄 cat

Ver conteúdo

cat arquivo.txt

🗑️ rm

Remover arquivo

rm arquivo.txt
Comparação Windows ↔ Linux 
Windows           Linux
dir          →    ls -la
cd           →    cd
mkdir        →    mkdir
del          →    rm
copy         →    cp
move         →    mv
type         →    cat
cls          →    clear

🎯 Exercício Final

🏆 Desafio: Ambiente Completo

  1. Instale o Windows Terminal via Winget
  2. Instale a fonte Cascadia Code
  3. Configure um tema escuro personalizado
  4. Instale o WSL com Ubuntu
  5. Crie uma pasta no Linux e navegue até ela
💡 Solução 
# 1. Instalar Windows Terminal
C:\> winget install Microsoft.WindowsTerminal

# 2. Instalar Cascadia Code
C:\> winget install Microsoft.CascadiaCode

# 3. Configurar tema (abra Terminal → Configurações → Aparência)

# 4. Instalar WSL
C:\> wsl --install

# 5. Após reiniciar, entrar no Linux
C:\> wsl

# No Linux:
user@DESKTOP:~$ mkdir meus-projetos
user@DESKTOP:~$ cd meus-projetos
user@DESKTOP:~/meus-projetos$ pwd
/home/user/meus-projetos
