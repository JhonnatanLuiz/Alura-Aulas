🔧 Variáveis de Ambiente

Variáveis de ambiente são valores armazenados pelo sistema operacional que podem ser usados por programas e scripts para obter informações sobre o ambiente de execução.

⏱️ Duração: 25-30 minutos | 📊 Nível: Intermediário

💡 O que são Variáveis de Ambiente

Variáveis de ambiente são configurações do sistema que armazenam informações importantes:

📂 PATH

Locais onde o Windows procura programas

👤 USERNAME

Nome do usuário atual

🖥️ COMPUTERNAME

Nome do computador

📁 TEMP

Pasta temporária

Ver todas as variáveis 
C:\> set

COMPUTERNAME=DESKTOP-ABC123
HOMEPATH=\Users\Voce
PATH=C:\Windows\system32;C:\Windows;...
TEMP=C:\Users\Voce\AppData\Local\Temp
USERNAME=Voce

🛣️ A Variável PATH

O PATH é crucial! Ela diz ao Windows onde procurar programas executáveis.

💡 Como funciona: Quando você digita python, o Windows procura em cada pasta do PATH.

Ver o PATH 
C:\> echo %PATH%

C:\Windows\system32;C:\Windows;C:\Python311\

⚙️ Comandos set e setx

set (Temporário)

Válido apenas na sessão atual

set VAR=Valor

setx (Permanente)

Salva permanentemente

setx VAR "Valor"
Exemplos 
# Temporária
C:\> set PROJETO=MeuApp

# Permanente
C:\> setx EDITOR "VS Code"

# Adicionar ao PATH
C:\> setx PATH "%PATH%;C:\MinhasPastas"

⚠️ CUIDADO: Sempre inclua %PATH% ao modificar PATH!

🐍 Python e PATH

Adicionar Python ao PATH 
# Localizar Python
C:\> where python
C:\Python311\python.exe

# Adicionar (como Admin)
C:\> setx PATH "%PATH%;C:\Python311;C:\Python311\Scripts"

# Verificar (nova janela)
C:\> python --version
Python 3.11.5

🎯 Exercício

🏆 Desafio

  1. Crie DEV_HOME para sua pasta de projetos
  2. Adicione uma pasta ao PATH
  3. Verifique com echo
💡 Solução 
C:\> setx DEV_HOME "C:\Projetos"
C:\> setx PATH "%PATH%;C:\MinhasFerramentas"
C:\> echo %DEV_HOME%

📌 Resumo

  • ✓ PATH define onde buscar executáveis
  • ✓ set = temporário | setx = permanente
  • ✓ Sempre inclua %PATH% ao modificar
