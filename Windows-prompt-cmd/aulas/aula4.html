🍫 Chocolatey - Gerenciador de Pacotes

Imagine instalar qualquer programa apenas digitando um comando. Isso é possível com gerenciadores de pacotes!

⏱️ Duração: 30-35 minutos | 📊 Nível: Intermediário

💡 O que é um Gerenciador de Pacotes?

Um gerenciador de pacotes automatiza a instalação, atualização e remoção de softwares:

❌ Modo Tradicional

  • • Buscar no Google
  • • Baixar instalador
  • • Clicar Next, Next...
  • • Configurar manualmente

✅ Com Chocolatey

  • • Um comando
  • • Instalação automática
  • • Atualização fácil
  • • Remoção limpa
Exemplo simples 
C:\> choco install git -y

Chocolatey v2.2.2
Installing the following packages:
git
...
git v2.43.0 has been installed successfully.

⚙️ Instalando o Chocolatey

⚠️ IMPORTANTE: Abra o PowerShell como Administrador!

Comando de instalação 
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

📋 Passo a passo:

  1. Clique com botão direito no ícone do Windows
  2. Escolha "Windows PowerShell (Admin)"
  3. Cole o comando acima e pressione Enter
  4. Aguarde a instalação
  5. Feche e abra novamente o terminal
Verificar instalação 
C:\> choco --version
2.2.2

⌨️ Comandos Principais

📥 choco install

Instala programas

choco install nome -y

🔄 choco upgrade

Atualiza programas

choco upgrade nome -y

🗑️ choco uninstall

Remove programas

choco uninstall nome -y

📋 choco list

Lista instalados

choco list --local-only

💡 Dica: O -y confirma automaticamente sem perguntar!

🚀 Exemplos Práticos

Instalando programas populares 
# Git (controle de versão)
C:\> choco install git -y

# Visual Studio Code
C:\> choco install vscode -y

# Python
C:\> choco install python -y

# Node.js
C:\> choco install nodejs -y

# Google Chrome
C:\> choco install googlechrome -y

# 7-Zip
C:\> choco install 7zip -y
Instalar múltiplos programas 
C:\> choco install git vscode python nodejs -y
Atualizar tudo 
C:\> choco upgrade all -y

🎯 Exercício

🏆 Desafio: Ambiente de Desenvolvimento

Crie um script setup-dev.bat que instale:

  1. Git
  2. Visual Studio Code
  3. Python
  4. Node.js
💡 Solução 
@echo off
echo Instalando ferramentas de desenvolvimento...
echo.

REM Instalar Git
echo [1/4] Instalando Git...
choco install git -y

REM Instalar VS Code
echo [2/4] Instalando VS Code...
choco install vscode -y

REM Instalar Python
echo [3/4] Instalando Python...
choco install python -y

REM Instalar Node.js
echo [4/4] Instalando Node.js...
choco install nodejs -y

echo.
echo Instalacao concluida!
pause

  • ✓ Atualize tudo com choco upgrade all -y
  • ✓ Sempre execute como Administrador
